# 🚀 UTP Match 

Bienvenido al repositorio oficial de **UTP Match**, la aplicación móvil diseñada para transformar la experiencia de orientación vocacional de los postulantes. A través de interfaces gamificadas, traducción de mallas curriculares con IA y comparativas interactivas, ayudamos a los estudiantes a tomar decisiones informadas y sin ansiedad.

---

## 🛠️ Requisitos Previos

Para probar la aplicación en tu propio dispositivo de manera nativa, asegúrate de tener instalado:

1. **Node.js** (Versión LTS recomendada).
2. La aplicación **Expo Go** instalada en tu celular (disponible gratis en iOS App Store y Google Play Store).

---

## ⚙️ Instrucciones de Instalación y Ejecución

Sigue estos dos sencillos pasos desde la terminal para levantar el proyecto:

### Paso 1: Instalar las dependencias
Abre tu terminal en la carpeta raíz del proyecto y ejecuta:

```bash
npm install
```

### Paso 2: Levantar el servidor
Abre tu terminal en la carpeta raíz del proyecto y ejecuta:

```bash
npx expo start -c
```
